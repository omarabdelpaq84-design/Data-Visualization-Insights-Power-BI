# 📊 Power BI Dashboard Project

## 📌 Overview

This project showcases an interactive Power BI dashboard designed to analyze data and present insights in a clear and visually appealing way. The dashboard enables users to explore trends, patterns, and key performance indicators بسهولة.

## 🛠️ Tools & Technologies

* Power BI
* Data Cleaning & Transformation (Power Query)
* Data Modeling
* DAX (Data Analysis Expressions)

## 📊 Features

* Interactive dashboard with filters and slicers
* Clear visualization of key metrics
* Easy-to-understand charts and graphs
* Dynamic insights for better decision-making

## 🔍 Analysis Process

1. Data Cleaning and Preparation using Power Query
2. Data Modeling and Relationships
3. Creating Measures using DAX
4. Building Interactive Visualizations

## 💡 Key Insights

* Identified major trends and performance patterns
* Highlighted key business metrics
* Provided actionable insights from the data

## 📁 Project Files

* `project1.pbix` → Power BI dashboard file

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Interact with the dashboard (filters, slicers, visuals)
